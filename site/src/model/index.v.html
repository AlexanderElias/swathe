Properties

controller

Parameters

  • name A controller name. Required
  • model A observed object. It is best to define all properties that will be observed for backwards browsers compatibility. Required
  • callback A callback for the controller with the instance of the controller passed as the first argument.

Returns

  • controller A controller instance.
